#### Node.js



```bash
# Установка
$ sudo apt update
$ sudo apt install nodejs
$ sudo apt install npm

# Проверка работоспособности
$ nodejs -v
$ npm -v

# создать проект Create React
$ npm install -g create-react-app

# Фикс ошибки с версией
$ npm cache clean -f 
$ sudo npm install -g n
$ sudo n latest

```
