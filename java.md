#### Java

```bash
# Установка Java
$ sudo apt update
# Установка JRE (последняя версия)
$ sudo apt install default-jre
# Установка JDK (последняя версия)
$ sudo apt install default-jdk

# Проверка установки Java
$ java -version

# Установка Maven
$ sudo apt install maven

# Проверка установки Maven
$ mvn -v
```
