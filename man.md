#### Man


```bash
# Вызов справки
$ man ls          # вызов справки о команде ls
$ man 2 write     # вызов справки о системном вызове write (из секции 2)

1 - пользовательские команды ОС Unix (ls, rm, mv);
2 - системные вызовы ядра ОС Unix;
3 - библиотечные функции языка C (sprintf);
4 - описания файлов устройств;
5 - описания форматов системных конфигурационных файлов;
6 - игровые программы;
7 - общие понятия (man 7 ip выдает справку о TCP/IP);
8 - команды системного администрирования OC Unix (mount).
```
