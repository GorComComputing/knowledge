Установка python:
```
$ sudo apt-get install python3
# Проверка установки python
$ python3 --version
$ sudo apt install python3-pip
$ sudo pip3 install ipython
# Проверка установки ipython
$ python3 -m pip show ipython
$ ipython --version
```
Python
```
$ python3
>>> exit()
```
Выполнение сценария Python:
```
$ touch hello.py
$ vi hello.py
$ python3 hello.py
```
IPython
```
$ ipython
In [1]: exit()
```
Jupiter Notebook
```
$ pip install jupyter
$ jupyter notebook
```
