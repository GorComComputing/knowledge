#### CMake

```
# Установка
$ sudo apt update
$ sudo apt install cmake
# Проверка установки
$ cmake --version


# Чтобы избежать конфликта между make и CMake можно переименовать исполняемый файл CMake
# Например, можно назвать его cmake или cm
$ sudo mv /usr/bin/cmake /usr/bin/cm

# Запуск CMake
$ mkdir build
$ cd build
$ cm ..
```
